# Web Components SEO tests

[![Netlify Status](https://api.netlify.com/api/v1/badges/c3e21651-7cb2-4469-b865-38ca8ac1381d/deploy-status)](https://app.netlify.com/sites/web-componenets-and-seo/deploys)
